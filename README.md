# Lando

We're using Lando to manage Docker containers.

Containers created:

`appserver` - PHP  
`appserver_nginx` - NGINX  
`database` - MySQL  
`node` - NodeJS
`mailhog` - MailHog local email server  
`pma` - phpMyAdmin

## Environment

Start: `lando start`  
Rebuild: `lando rebuild`

To retrieve local URLs, port numbers
